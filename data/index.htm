ZX Browser

This browser will allow a download to a ZX80 or ZX81 using the cassette audio input interface. The functions available are fairly simple and are basically part of the original FSBrowser program for the 8266 that is the basis for this program. Invoke the browser using the 'IPaddr'/edit or click this link: edit. This will kick off the editor. Files available will be listed on the left. By selecting a .80 or .o file, you will initiate the sending of the file to the ZX80 equiped with a 4K ROM. Selecting a .p, .81 or .p81 will commence sending a file for an 8K ROM equiped ZX80 or for a ZX81. Note that the programs for the 4K ROM are not compatible with the 8K ROM and visa versa. Please note that this can take some time, since the output to the ZX80 is only about 300 bits/sec. Be patient. The 8266 module LED will flash during the download. There is a file with a basic explanation of the hookups here: instructions .
